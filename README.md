# Overview

This project is about bike trip analysis. The analysis use visualization to show:
- Show the length of time that bikes are checked out for all riders and genders
- Show the number of bike trips for all riders and genders for each hour of each day of the week
- Show the number of bike trips for each type of user and gender for each day of the week.

# Results
- Most trips are within 20 minutes
- Men prefer to ride bicycles than women, but trip duration similar.
- On workday, peak is commute time.
- Men more like ride bicycles than women, and peak tiem similar.
- Trips of workday are bigger than weekend.
- There are 13,983 bikes. There are 2,344,224 trips in August 2018.
- Some bikes were rided 479 times in August 2018.
